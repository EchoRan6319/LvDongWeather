# 更新日志

## v4.0.0

- **应用升级**：版本号升级至 4.0.0
- **国际化支持**：完整支持中文和英文切换
- **Windows 字体优化**：Windows 端强制使用微软雅黑字体，解决中文字体显示不一致问题
- **天气助手改进**：
  - 英文模式下显示 AI 英文支持提示
  - 优化英文输出格式，修复单词粘连问题
  - 移除 AI 回答中的货币符号
- **天气数据本地化**：天气状况和风向支持英文显示
- **UI 统一**：语言选择页面 UI 与其他设置页面保持一致
- **语言切换优化**：切换语言后无需重启应用即可生效

## v3.7.0

- **动态颜色优化**：修复 Windows 上动态壁纸取色不更新的问题
- **跨平台兼容**：所有平台正确响应壁纸变化
- **用户体验**：应用从后台返回前台时自动更新动态颜色

## v3.6.0

- **字体移除**：移除 OPPO Sans 字体，使用系统默认字体
- **代码清理**：简化字体相关配置

## v3.5.0

- **图标适配**：为所有平台添加统一图标
- **构建优化**：成功构建 Windows 平台应用

## v3.4.0

- **天气助手优化**：添加自动换行功能，提高文本可读性
- **构建优化**：支持从环境变量读取 API Key
- **稳定性改进**：增强应用启动时的错误处理

## v3.3.0

- **功能完善**：重构天气数据展示方式，为 AI 助手提供更详细的数据

## v3.2.0

- **布局优化**：顶部天气信息和天气详情卡片集成在一个页面
- **本地化修复**：修复长按操作名称未汉化的问题
- **用户体验**：Windows 端支持鼠标拖拽滚动

## v3.1.0

- **卡片顺序修复**：修复天气详情页卡片排序功能
- **默认顺序优化**：设置默认卡片顺序
- **顺序验证**：自动修复错误的保存顺序

## v3.0.0

- **图标更新**：全新 Material You 风格应用图标
- **界面优化**：整体 UI 风格现代化
- **平台支持**：增加 Windows 平台支持

## v2.6.0
